# LÛTKE — Ses Dosyaları

## Format
- **Codec:** OPUS (Opus Audio Codec)
- **Örnek hızı:** 44.1kHz
- **Bit derinliği:** 16-bit
- **Seviye:** Ambient < 30dB
- **Referans:** DLIFLC (ABD Savunma Dil Enstitüsü) standardı

## Klasör Yapısı

```
assets/audio/
├── a1/
│   ├── a1_001.opus    → "Roj baş" (Günaydın)
│   ├── a1_002.opus    → "Spas" (Teşekkür)
│   └── ...
├── a2/
├── b1/
├── b2/
├── c1/
└── c2/
```

## Kayıt Standardı

Her kelime için:
1. **Yavaş** — Telaffuz açık, net (öğrenci için)
2. **Normal** — Doğal konuşma hızı
3. **Cümle** — Kelime cümle içinde

## Anadil Konuşucu Gereksinimleri

- Kurmancî anadil konuşucu (Kuzey lehçesi)
- Yaş: 30-50 (standart telaffuz)
- Bölge: Tercihen Amed (Diyarbakır) veya Riha (Şanlıurfa) ağzı
- Standart Kurmancî (dialekt nötür)
